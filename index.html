    2- Para que se usa reverse en un array
    Reverse se usa para para revertir un elemento dentro de un arreglo. Ejemplo
    let texto = ["hola", "Chau"]; const reversive = () => { return texto.reverse(""); }; console.log(reversive()); Mostrará por consola (Chau, hola)
    3- Enumera los tipos de funciones que se usan en js
    Funciones anónimas (anonymous function)
    const anonimo = function () { console.log("Funcion anónima"); };
    Funciones declaradas (named function)
    function declarada() { console.log("Funcion declarada"); }
    Funciones Flechas (arrows function)
    const arrow = () => { console.log("Funcion flecha"); };
    4- Qué es una arrow function
    Una función de flecha es una sintaxis más corta para una expresión de función y no tiene su propio this, argumentos, super, o new.target Estas funciones son más adecuadas para funciones que no son métodos y no pueden utilizarse como constructores.
    5- Explique hoisting en js
    hoisting es un comportamiento por defecto de js donde todas las declaraciones de variables y funciones se mueven a la parte superior. esto significa que independientemente de donde se declaren las variables y las funciones, se mueven en la parte superior del ámbito, que puede ser tanto local como global.
    6- Que es la propiedad NaN en javascript
    NaN representa "Not a Number", indica que el valor no es un número legal
    7- Cuales son algunas de las ventajas de usar javascript externo

    Constructor function se usa para crear objetos en js, cuando se usan? para cuando queremos crear multiples objetos que tienen similares propiedades y métodos
